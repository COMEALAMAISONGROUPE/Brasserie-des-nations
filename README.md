# Brasserie des Nations

Site vitrine officiel de la **Brasserie des Nations**, restaurant traditionnel luxembourgeois depuis 1956, situe au coeur de Luxembourg-Ville.

## Apercu

- Cuisine luxembourgeoise traditionnelle (Judd mat Gaardebounen, Bouchee a la Reine, Cordon Bleu...)
- Menu du jour, carte des vins, specialites locales
- Ambiance brasserie authentique au coeur de la Ville Haute

## Technologies

- HTML5 / CSS3
- JavaScript vanilla
- PWA (Progressive Web App) avec manifest.json
- Heberge sur GitHub Pages

## Structure du projet

```

|-- index.html                    # Page d'accueil principale

|-- manifest.json                 # Manifest PWA

|-- css/

|-- js/

|   |-- video-bg.mp4              # Video de fond hero
```
